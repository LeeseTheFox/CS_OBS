# CS_OBS

CS_OBS is a simple tool that automatically controls OBS Studio's Replay Buffer based on which games you're playing. If you're looking for an alternative to Nvidia ShadowPlay that gives you more control over when your recordings start and stop, CS_OBS might be just what you need. The code was entirely generated by Claude and ChatGPT, so judge accordingly.

## What does it do?

- Starts OBS and its Replay Buffer when you launch specific games
- Closes OBS when you're done playing to free up your computer's resources
- Works only with games you choose (no unwanted recording)
- Runs quietly in your system tray until needed
- Can be paused when you don't want it to run
- Can start with Windows if you want

## Prerequisites

Before using CS_OBS, ensure that you have .NET 6.0 installed on your system. You can download it from the official .NET website:

- [Download .NET 6.0 Runtime](https://aka.ms/dotnet-core-applaunch?missing_runtime=true&arch=x64&rid=win10-x64&apphost_version=6.0.23&gui=true)

## Setting it up

The first time you run CS_OBS, it will ask you to:

1. Specify the location of your OBS Studio installation
2. Add the games you want it to work with
   - You can pick from games that are currently running
   - Or type in the name manually if the game isn't running

## Adjusting settings
![Settings Image](https://github.com/LeeseTheFox/CS_OBS/raw/master/Images/Settings.png)

Right-click the CS_OBS icon in your system tray and press 'Settings' to:

- Set up your OBS location
- Add or remove games from your list
- Change how quickly OBS starts and stops
- Choose whether you want CS_OBS to start with Windows

## Using CS_OBS

After setup, CS_OBS works automatically:
1. When you start a game from your list, it opens OBS
2. When you close your game, it closes OBS
3. That's it!

## Good to know

- CS_OBS needs admin rights to function properly
- Make sure you've set up OBS Studio's Replay Buffer as desired
- Your settings are saved in your AppData folder (located at `C:\Users\<YourUser>\AppData` on Windows)
